# TCP #
FTP (21) - File Transfer Protocol - A server to upload and download files

SSH (22) - Secure Shell - Remote Shell into a machine

Telnet (23) - Old unencrypted way to log in remotely to a machine

SMTP (25) - Mail

DNS (53) - Domain Name System - Resolve IP addresses to names/domains

HTTP (80)/HTTPS (443) - HyperText Transfer Protocol (Secure if using HTTPS) - Websites

POP3 (110) - Mail

SMB (132 + 445) - File Shares/Samba - Golden Standard of Exploits

IMAP (143) - Mail

# UDP #
DNS (53) - Domain Name System - Resolve IP addresses to names/domains (it is both a TCP and UDP protocol)

DHCP (67, 68) - Associates users with IP adresses at random 

TFTP (69) - Trivial FTP - Same as FTP but uses UDP instead of TCP

SNMP (161) - Simple Network Management Protocol
