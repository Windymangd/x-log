Look at ip configuration
`ipconfig` for windows
`ifconfig` for unix-based

It will show both the current machine's ipv4 and ipv6 by default.
ipv4 uses decimal notation
ipv6 uses hexadecimal

Every segment of ipv4 is 1 byte or 8 bits, making the whole 4 bytes.

NAT - Network Address Translation
Private ips are assigned within networks.

Private IP ranges are:
10.0.0.0-10.255.255.255 :: CLASS A
172.16.0.0-172.31.0.0 :: CLASS B
192.168.0.0-192.168.255.255 :: CLASS C
Localhost/Loopback-127.0.0.0-127.0.0.7 (usually 127.0.0.1 on most systems)
